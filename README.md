## Dependencies:

It uses `md2resume` from [there4/markdown-resume](https://github.com/there4/markdown-resume).

It does depend on `wkhtmltopdf`. Install it with `brew` on OSX:
```sh
$ brew install homebrew/boneyard/wkhtmltopdf
```
